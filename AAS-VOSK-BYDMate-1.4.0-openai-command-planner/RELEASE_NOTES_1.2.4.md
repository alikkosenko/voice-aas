# AAS-VOSK-BYDMate 1.2.4

## Новые команды

- «Вентиляция всех сидений 1/2/3»
- «Обдув всех сидений минимум/средний/максимум»
- «Обогрев всех сидений 1/2/3»
- «Подогрев всех сидений минимум/средний/максимум»
- «Выключи вентиляцию всех сидений»
- «Выключи обогрев всех сидений»
- «Подогрев руля 1/2/3»
- «Подогрев руля минимум/средний/максимум»
- «Выключи подогрев руля»

Команда «всех сидений» управляет передними водительским и пассажирским сиденьями. Задний ряд не добавлялся, поскольку в текущем каталоге отсутствуют подтверждённые BYD-параметры.

## Изменения

- Интенсивность сидений и руля ограничивается диапазоном 1–3.
- Числа распознаются цифрами и словами: один/первый, два/второй, три/третий.
- Добавлены словесные уровни: минимум, средний, максимум.
- Для всех сидений команда последовательно применяется к водительскому и пассажирскому местам.
