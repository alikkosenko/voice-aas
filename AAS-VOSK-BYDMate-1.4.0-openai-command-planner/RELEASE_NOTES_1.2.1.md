# AAS 1.2.1

- Исправлен приоритет разбора команды «обдув сиденья»: она больше не воспринимается как скорость вентилятора салона.
- Расширены варианты фраз для подогрева и вентиляции водительского/пассажирского/всех сидений.
- Для сидений сначала отключается противоположный режим, затем отправляются оба известных BYD-пути управления: switch/level и command fallback.
- Исправлено значение выключения прямого switch-параметра с `2` на `0`.
- Холодильник автоматически выбирает охлаждение для −6…15 °C и нагрев для 35…50 °C.
- Диапазон охлаждения расширен до 15 °C. Значения 16…34 °C отклоняются с понятным сообщением.
