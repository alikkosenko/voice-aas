# AAS-VOSK-BYDMate 1.2.7

## YouTube ReVanced

- добавлена команда поиска по любой строке после слова `YouTube`/`Ютуб`;
- добавлены русские и украинские формы `Найди/Знайди на YouTube ...`;
- выбранный в настройках YouTube/ReVanced package имеет приоритет;
- поиск выполняется через `ACTION_SEARCH`, с резервным переходом на URL результатов;
- добавлены fallback package ID для ReVanced и ReVanced Extended;
- справочник команд обновлён на русском и украинском.

Версия: `versionCode 127`, `versionName 1.2.7`.
