# AAS-VOSK-BYDMate 1.2.6

## Украинский язык

- Добавлено полноценное распознавание естественных команд на украинском языке во всех поддерживаемых разделах.
- Добавлены украинские числительные, формы команд, навигационные фразы и названия зон автомобиля.
- Все основные результаты выполнения, ошибки, состояния оверлея и служебные сообщения переводятся на украинский при выборе `uk-UA`.
- В APK по-прежнему используются отдельные офлайн-модели Vosk для русского и украинского языков.

## Список команд

- Окно «Все голосовые команды» разделено на категории: климат, сиденья, окна/люк, двери/багажники, освещение, холодильник, данные автомобиля, мультимедиа и навигация.
- В диалоге можно переключаться между русским и украинским списком.
- Внизу отображаются все 50 импортированных `WriteAllowlist` action_name, сгруппированные по категориям, с допустимыми диапазонами значений и отметкой непроверенных fallback-каналов.
- Обновлён корневой `COMMANDS.md` с полным двуязычным справочником.

## Озвучивание ответов

- Исправлена причина молчания после отключения штатного ассистента: helper больше не отключает пакет `com.byd.autovoice.tts` и умеет принудительно включать его обратно.
- При восстановлении runtime сервис автоматически включает TTS-пакет и пересоздаёт `TextToSpeech`.
- AAS сначала пробует штатный BYD TTS, затем системный TTS по умолчанию.
- В AndroidManifest добавлены package visibility-запросы для BYD TTS и `android.intent.action.TTS_SERVICE`.
- Для речи задаются атрибуты навигационных подсказок и поток мультимедиа.
- Добавлены очередь ожидающей фразы, повторная инициализация при ошибке и кнопка «Проверить озвучивание».
- Изменение языка пересоздаёт TTS с нужной локалью.
