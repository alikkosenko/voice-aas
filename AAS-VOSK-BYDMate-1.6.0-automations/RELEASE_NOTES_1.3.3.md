# AAS 1.3.3

## Исправление голосовых ответов

- Офлайн-подтверждения команд теперь воспроизводятся прежде всего через обычный медиаканал Android (`STREAM_MUSIC`).
- BYD BTTS stream 17 оставлен только резервным маршрутом: на части DiLink он создавался успешно, но не выдавал звук.
- Добавлен кратковременный аудиофокус для голосового подтверждения.
- `AudioTrack` переведён в потоковый режим с полной проверкой записи PCM.
- Сохранены автономные RU/UA-фразы: успешная команда и ошибка работают без установленного TTS-движка.
- HTTP-проверка пароля и исправления runtime из 1.3.2 сохранены.
