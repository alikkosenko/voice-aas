# AAS 1.2.0

- Переключатель «Озвучивать ответы голосового ассистента» сохранён в настройках и теперь снабжён явным описанием.
- При выключенной озвучке команды продолжают выполняться, но TTS не произносит ответы.
- Оверлей выводится в приоритетную очередь UI сразу после нажатия кнопки, до подготовки AudioRecord/SpeechService.
- Добавлены состояния оверлея: «Подготавливаю…», «Слушаю…», «Распознаю…», «Выполняю…».
- Оверлей больше не исчезает между окончанием записи и выполнением команды.
- Добавлена короткая анимация появления 120 мс.
